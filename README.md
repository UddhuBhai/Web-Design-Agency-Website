# 🎨 Web Design Agency Website

A modern and responsive landing page for a fictional web design agency, built using HTML and CSS. This project focuses on creating a clean layout while implementing responsive design principles for a seamless experience across different devices.

## 📖 About

This project showcases a simple agency website featuring a professional landing page, service cards, typography, images, and a responsive layout. To add a fun personal touch, I incorporated references from two of my favorite anime series:

* 🌸 **Boa Hancock** from **One Piece** represents the **Beauty** section, symbolizing elegance, charm, and visually stunning designs.
* ⚙️ **Edward Elric** from **Fullmetal Alchemist** inspires the **Construction** section, representing creativity, precision, and building strong, reliable websites.

These references make the project more unique while blending pop culture with web design.

## 🚀 Features

* 🎨 Clean and modern UI
* 📱 Responsive design for mobile and desktop
* 🖼️ Image-based service cards
* ✨ Custom typography with Google Fonts
* 📐 CSS Float-based layout
* 📲 Media Queries for better responsiveness
* 🎌 Creative anime-inspired content integration
